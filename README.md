This repository contains three MATLAB functions to define ROI, measure MFI and format acquired data designed for fluorescent microarray readout from uMAP.

roi_grid_measure_tool: a GUI to define ROI to measure pixel intensities from a fluorescence image of a single microarray
<img width="1456" height="870" alt="fig9 umap ROI matlab" src="https://github.com/user-attachments/assets/2cba9f4b-a6af-45e2-a32a-f025170b9d70" />
  input: execute the function, then select a fluorescent image of a uMAP microarray by using the load image button
  
  operation: use the buttons on the GUI to adjust the layout of ROI, then use export to excel button to create output
  
  output: an excel file containing mean and median of each ROI with its index of row and column

  

Combine_Median: a code to combine medians from excel files generated by roi_grid_measure_tool
  input: automatically selects all excel files in the same directory as the function
  
  operation: adjust the parameters in the code based on the design of the microarray, then execute the function. Parameters include:
    antigen_amount: number of antigens in microarray;
    antigen_name_row_index = 9;
    MFI = 10;
    BSA_column_index: the column index of BSA to be used for background subtraction;
    Perform_BSA_subtract: 1 as true, 0 as false;
  
  output: an excel file containing processed median
